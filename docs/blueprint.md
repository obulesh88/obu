# **App Name**: EarnEasy

## Core Features:

- Task Listing: Display a list of available tasks with their respective reward amounts.
- Task Completion: Allow users to mark tasks as complete and submit evidence for verification.
- Reward Tracking: Track users' earnings and display their balance.
- Payment Processing: Enable users to withdraw their earnings via connected payment methods.
- Task Recommendation: The app uses an AI tool to recommend tasks to users based on their historical submissions and behaviors, so they'll spend less time searching.

## Style Guidelines:

- Primary color: Light blue (#64B5F6) for trust and reliability.
- Background color: Very light blue (#E3F2FD), near white, for a clean interface.
- Accent color: Yellow (#FFCA28) to highlight earning opportunities and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif, for clear and accessible readability.
- Use simple, clear icons to represent task types and reward amounts.
- Prioritize a clean, uncluttered layout with a clear hierarchy of information.
- Use subtle animations to indicate task completion and reward confirmation.
